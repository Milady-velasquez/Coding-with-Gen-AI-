# Coding-with-Gen-AI-
##Feedback from students

##### Problem

I want to create a solution for a college course feedback system that allows students to submit feedback for their classes using their school email addresses. The program should verify that the email belongs to a registered student before accepting the feedback and prevent students from submitting more than one response for the same course. If the student email is not found in the list of registered students, the progeam should display "Invalid Email" message. and if the student attempts to submit feedback more than once, the system should tell them that they have already submitted a response. The program should have Object-Oriented Programming principles by organizing the solution into multiple classes, also use a HashSet data structure to efficiently track submitted email addresses and detect duplicate submissions.
